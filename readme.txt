== Description ==

WPLingo is a lite weight plugin which allows to build classic or modern forum site in seconds. It is super simple to configure and will work with ANY properly coded WordPress theme and you can use it on new or existing site.

When building Lingo we are focusing on ease of use, WordPress compatibility and extensibility.

= Main Features =

* Responsive design.
* Easy to install, configure and use.
* Quickly moderate threads with fast AJAX based moderation tools.
* Powerfull access restriction options.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wplingo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the: wp-admin -> Plugins screen in WordPress
3. On activation plugin will create three Pages (in wp-admin / Pages panel) with [lingo_forum], [lingo_user_page] and [lingo_search] shortcodes .
4. Go to: wp-admin -> Forums -> Options. Click "Settings" in Core module tab and configure options. Be sure that You set proper pages in "Pages & URLs" section!
